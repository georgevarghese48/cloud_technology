Docker Tutorial


Step 1. Installing Docker

You can use either of the 2 links provided below to install Docker on your machine.

->Link 1

->Link 2

Step 2. Pull an image from Dockerhub

After you've installed the application and have logged in, you should open up your command prompt and run the command: docker pull ubuntu

To check that you've successfully pulled the image, run the command "docker image", which should show you information regarding the image.

Step 3. Committing Changes to a Docker image

First, you should run the Ubuntu image, which would start a Docker container.

The command for this is: docker run -it ubuntu:latest /bin/bash

Next, make sure you have everything you need installed and or updated.

Run these commands in order:

-> apt-get update

-> apt-get install apache2

-> apt-get install vim

Once you're done installing and updating, you can type "exit" into the command line and enter, to exit the container.

Finally, you'll want to check the containers information using the command: docker ps -a

You'll run this next command with some of the information you gathered from the previous command mentioned, the command is:

-> docker commit (containerID)(name of new image)

Step 4. Sharing Data between a Container and Host

For this next step, before you do anything, you should make sure that no other containers are using port 80 for the sake of this tutorial. If they are, use the command "docker stop (containerID)" with the container ID being the one of the container using port 80.

Once this is done or if no container was using port 80, you can access the interactive shell. The next command will be dependant on the pathing of your files on your machine, so here is an example of what the command should look like:

-> docker run -it -p 127.0.0.1:80:80 -v C:\Users\Student1:/var/www/html (name of new image) /bin/bash

Step 5. vi Editor in UNIX

Now that you are in the instance, type the command "cd /var/www/html" to travel to the html directory.

Once in this directory, you should create an index.html file by typing the command "vi index.html", which will simultaneously create the file and put you in vi Editor. Now that you are in editor, you'll want to press the "i" key on your keyboard, which will prompt "--INSERT--" to pop up at the bottom of the window, which signifies that you are now permitted to edit the file with html source code now.

When you are finished editing the file, you can exit the editor by inputting the following commands:

-> Press the "Esc" key on your keyboard to exit out of Insert mode

-> Type ":wq" to write the file, and then quit vi Editor.

Step 6. Using Docker to host a website

In your command prompt and while still in Docker container instance, type in the following command to start the apache service:

-> /etc/init.d/apache2 restart

Next, all you have to do is type in the IP address previously used in step 4, "127.0.0.1", into a browser of your choice. Once you press enter you should be brought to your site.
